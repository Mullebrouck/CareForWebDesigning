Abbreviation Name Description
RGB Red, Green and Blue RGB colors are used for web application such as websites, etc. They are also referred as "safe" colors.
CMYK Cyan, Magenta, Yellow, and Black CMYK is also called the 4-color process for printing. The printer prints these 4 colors, that are indicated by % in small dots. To be able to see the full spectrum of the color or eyes blend the small dots together.
PMS Pantone Matching System This matching system contains thousands of numbered swatched owned by Pantone. These colors are used for printing.
HSB Hue, Saturation, and Brightness This color model is represented by 3 numbers. The value of Hue is set in degrees, the value of saturation in percentages and brightness ranges from 0 - 100.
HEX Hexadecimal The HEX number is a six-digit, three-byte number that represents a color. It is commonly used in HTML, CSS, SVG, ...
